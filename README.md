# Queue
Queue is mobile first web application allowing user to book their slot to effectively plan their time.

It is nextjs best mobile first web application 

Feature Of this application.
1. User of the application let say barber shop or any such place where waiting is possible need to book an appointment
2. User needs to know when their turn will come so that they can plan and actively notified their turn in whatsapp
3. User can easily confirm based on current status or queue along with estimated time to confirm it via whatsapp chatbot or similar prompts based on yes no
4. This application is useful when customer already know the product but needs the estimated time of the turn.
5. Vendor person which actually operates the shop would need to provide the list of services they provide and estimated approx time it takes for them to get it done.
6. Predictive analysis should be used when such service is consumed by customers so that accuracy can be increase.
